
# React-Weather

* Ссылка на Netlify: https://react-weather-80152c.netlify.app

## Краткое описание проекта

Сайт, показывающий текущую погоду в выбранном городе

## Технологии, которые использованы в проекте

- React
- TypeScript
- Sass

## Установка и запуск

### Установка

```bash
npm install
```

### Запуск проекта:

```bash
npm start
```

### Запуск тестов:

```bash
npm run test
```
